# irb.wasm

IRB on browser powered by WebAssembly.
You can try irb.wasm on your browser. It works on CRuby ported to WebAssembly.

Demo: https://irb-wasm.vercel.app/

## Development

```console
$ rake static/irb.wasm
$ npm install
$ npx parcel ./src/index.html
```
